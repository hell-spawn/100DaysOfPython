## Data types 

- String =>  "Hello" 'Hello'
- Integer => 123 123_456
- Float => 3.14159 765_432.12
- Boolean => True , False

# Function Type 
```python
    print(type(123))
    >> <class 'int'>
```

# Functions  str() int() float() converters

```python
    print(type(str(123)))
    >> <class 'str'>
```

# Sub-string

```python
    print("Hello"[0])
    >> H
```
